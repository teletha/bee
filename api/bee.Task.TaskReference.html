Task.TaskReference<T>

  • T

    The type of the Task interface containing the referenced command.

  • ConsumerT
  • Serializable
  • ValuedTaskReferenceT, R

Represents a type-safe reference to a task command, typically implemented using a method reference (e.g., MyTask::myCommand).

This functional interface serves as a marker and must be Serializable to allow Bee to potentially introspect the reference (e.g., using TaskInfo#computeTaskName(TaskReference)).

It extends Consumer to indicate that it accepts a task instance to operate on.
