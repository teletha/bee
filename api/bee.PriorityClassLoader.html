A ClassLoader that prioritizes loading specified classes from its own classpath (URLs). For other classes, it follows the standard parent-first delegation model.

The binary name of the class to load.
booleanresolve

If true, resolve the class after loading.
Class?

ClassNotFoundException

Loads the class with the specified binary name. If the class name is in the highPriorityClasses set, it attempts to load the class from this loader's URLs first. Otherwise, it follows the standard parent-first delegation model.
