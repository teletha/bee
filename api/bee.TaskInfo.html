TaskInfo

Represents metadata about a task, including its computed name, defining class, available commands, and command descriptions. This class provides static methods to find and manage task information.

toString()String

String

A formatted string (e.g., &quot;compile \tCompile source files&quot;).

Returns a simple string representation of this task, typically used for listing available tasks. It includes the task name and the description of its default command.

find(ClassTsupport)T

TTask

The type of the task.
ClassTsupport

The class type of the task to find.
T

An executable (potentially proxy) instance of the specified task type.

Finds and returns an executable instance of the specified task type. This method retrieves the TaskInfo for the class and creates a proxy task.
