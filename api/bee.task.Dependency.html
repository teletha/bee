Dependency

Provides tasks for analyzing and displaying project dependencies. This includes showing the dependency tree and analyzing required Java modules.

tree()void

Displays the project's dependency graph in a tree format. This command visualizes the direct and transitive dependencies of the project, showing how libraries depend on each other. This is the default command for the dependency task.

module()ListString

ListString

A list of required module names.

Analyzes and displays the Java modules required by the project and its runtime dependencies. Uses the jdeps tool to analyze bytecode and also considers require-module entries in the MANIFEST.MF of dependency JARs. Requires the main sources to be compiled first.
